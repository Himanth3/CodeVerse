# CodeVerse — Project Documentation
## Project Name: CodeVerse

## Project Overview
CodeVerse is a full-stack Learning Management System (LMS) web application built for aspiring developers.
It offers interactive coding courses, quizzes, downloadable resources, and a personalized learning experience
— all wrapped in a modern, space-themed user interface with glowing accents and dynamic animations.

The platform empowers students to learn technologies like Java, Python, Spring Boot, Git & GitHub,
REST APIs, and Data Structures & Algorithms — all in one place.

## About This Website

### 🌌 What is CodeVerse?
CodeVerse is an online coding education platform designed to make programming approachable and engaging.
The platform combines a sleek, space-themed UI with practical coding lessons, giving learners an
immersive learning experience.

### 📚 Courses Offered
1. **Core Java Programming** — Variables, OOP, packages, exception handling, multithreading
2. **Java + DSA** — Arrays, Linked Lists, Trees, Graphs, Sorting, Dynamic Programming
3. **Git & GitHub Mastery** — Version control, branching, merging, pull requests, open source
4. **Building REST APIs** — HTTP methods, JSON, Spring Boot APIs, Postman testing
5. **Basic Python** — Variables, loops, functions, file handling, data structures
6. **Spring Boot Backend** — DI, JPA, Spring Security, JWT, microservices, deployment

### 🖥️ Key Features
- **User Authentication** — Secure Signup & Login using JWT tokens
- **Course Catalog** — Browse and explore multiple programming courses
- **Course Detail Pages** — Topics list, code examples, and learning objectives
- **Quizzes** — Interactive quizzes to test knowledge after each lesson
- **Downloads** — Downloadable resources, cheat sheets, and study materials
- **Projects Showcase** — A dedicated section for student and demo projects
- **Contact Page** — Reach out form for support and inquiries
- **User Profile** — Personalized profile page for authenticated users
- **Light / Dark Mode** — Persistent theme switching across the entire platform
- **Space-Themed UI** — Glowing accents, floating animations, and a cinematic look

## Development Process

### 1. Frontend Setup
- Initialized using Vite: `npm create vite@latest`
- Project structure: `src/pages/`, `src/components/`, `src/api/`, `src/data/`
- Routing configured with React Router Dom (v6)

### 2. Backend Setup
- Initialized via Spring Initializr
- Package: `com.codeverse.backend`
- REST API served at `http://localhost:8080`

### 3. Frontend–Backend Integration
- Spring Boot CORS configured to accept requests from Vite dev server (`localhost:5173`)
- Axios configured with a base URL pointing to the Spring Boot API

### 4. Authentication Flow
1. User submits credentials via Login/Signup page
2. Spring Security validates and returns a **JWT access token**
3. React stores the token in `localStorage`
4. Token is attached to all subsequent protected API requests via Axios interceptors

### 5. UI & Theming
- CSS variables used for a full light/dark mode system
- Framer Motion powers all page transitions and element animations
- Space/cosmic theme with neon glow effects, star-like particles, and gradient backgrounds
